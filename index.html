n8nเกี่ยวกับ n8n & สถาปัตยกรรมAbout n8n & its architecture

n8n คือแพลตฟอร์ม Workflow Automation แบบ open-source (fair-code) ที่ self-host ได้เอง — ต่อ API/บริการกว่า 400 ตัวผ่าน Visual Node Editor และเขียน Code Node (JS/Python) แทรก business logic ได้อิสระ ผมรันแบบ self-hosted บน VPS เอง และใช้ Queue Mode (Redis) เพื่อกระจายงานหลาย worker ให้รองรับโหลดสูง · ที่ใส่ stack ด้านล่างเพราะเข้าใจถึงระดับ internals — แก้ปัญหา/เขียน custom node/ปรับ performance ได้ ไม่ใช่แค่ลากวาง n8n is an open-source (fair-code) workflow-automation platform you can self-host — wiring 400+ APIs through a visual node editor while dropping in Code nodes (JS/Python) for custom business logic. I run it self-hosted on my own VPS in queue mode (Redis) to spread work across workers under load. I include its stack below because I understand it down to the internals — debugging, writing custom nodes and tuning performance, not just drag-and-drop.
